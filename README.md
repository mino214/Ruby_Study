## RUBYを勉強するリポジトリです。

## 1. Rubyとは？
> Rubyは、シンプルで読みやすい文法が特徴のプログラミング言語です。  
> Web開発（Ruby on Rails）をはじめ、スクリプト作成や自動化、学習用にも広く使われています。  
> このリポジトリでは、基礎文法・入出力・条件分岐・繰り返し・メソッドなどを、コードを書きながら学んでいきます。

- [Ruby公式サイト](https://www.ruby-lang.org/ja/)
- [Ruby Web Editor (Try Ruby)](https://try.ruby-lang.org/)

## 2. Ruby_Study リポジトリについて
> Rubyの基礎を学ぶための学習用リポジトリです。  
> 学習用の `Introduction` と、React + Ruby on Rails を使った `Ruby_Projects` に分けて管理しています。

### ディレクトリ構成（例）
- `Introduction/` : Ruby基礎（文法・入出力・例外処理など）
- `ruby_on_rails` : Ruby on Rails基礎
- `Ruby_Projects/` : React + Ruby on Rails を使った開発・練習
