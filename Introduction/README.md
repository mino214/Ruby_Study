## 学習順（おすすめ）

1. `HelloRuby.rb`
   - Rubyの実行に慣れる（基本の出力・最初の一歩）

2. `dataType.rb`
   - 基本データ型（数値／文字列／配列／ハッシュ など）と基本操作

3. `if_loop_1.rb`
   - 条件分岐（if）＋繰り返し（loop）入門

4. `if_loop_2.rb`
   - 条件分岐・繰り返しの応用（パターンを増やす）

5. `method.rb`
   - メソッド定義／引数／戻り値／スコープの基礎

6. `block_enumerable.rb`
   - ブロック（block）と Enumerable（each / map / select など）

7. `class_object.rb`
   - クラスとオブジェクト（初期化、インスタンス変数、メソッド）

8. `module.rb`
   - モジュール（include / extend、名前空間、Mixin）

9. `require_relative.rb`
   - ファイル分割と読み込み（require_relative）で構成整理

10. `file_exception.rb`
   - ファイル入出力（File I/O）＋例外処理（begin/rescue/ensure）

11. `lambda_callback.rb`
   - callback処理(Proc/lambda/ブロック)

---

### 実行方法
```bash
ruby HelloRuby.rb
ruby dataType.rb
# ...
